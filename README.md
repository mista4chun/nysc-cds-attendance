## 2026-07-15 — Mobile freeze fix

**Problem:** PWA would freeze/hang on random tabs (e.g. Reports) after phone
was locked/idle for hours. Fixed by logging in fresh, but bug would return.

**Root cause:** Supabase JWT expires after 1hr. autoRefreshToken only works
while JS timers are running — phone sleep suspends the tab entirely, so no
refresh happens. On wake, first request needing a valid token just hangs.

**Fix:** components/layout/AppResumeHandler.tsx — listens for
visibilitychange, proactively checks/refreshes session when app resumes,
redirects to /login if refresh fails instead of hanging.
